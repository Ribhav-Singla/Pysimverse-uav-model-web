A MuJoCo-based simulation environment for visualizing UAV navigation trajectories across different agent types and obstacle configurations.

Compare the performance of AR-PPO, Neurosymbolic, and Pure Neural agents as they navigate through environments with varying obstacle densities (1-15 obstacles).

Getting Started

  1. Select an Agent Type from the dropdown
  2. Choose the Obstacle Count (1-15)
  3. Click Load Scene to visualize the environment
  4. Click Run Simulation to see the UAV trajectory

Green dots will mark the path taken by the UAV during simulation.
